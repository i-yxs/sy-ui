SYUI
===========================
兼容全端的UI组件库项目

使用方法：
---------------------------
1、拉取仓库代码到本地
2、使用命令行运行以下命令安装依赖
```js
npm i
```
3、使用HBuilder X导入项目（文件-导入-从本地目录导入）<br>
4、使用HBuilder X运行项目（运行-选择需要运行的平台）<br>
5、使用HBuilder X发布项目（运行-选择需要发布的平台）<br>
